JSX подробнее

Фундаментально, JSX всего лишь предоставляет синтаксис для функции React.createElement(component, props, ...children)

Например, данный JSX код


        <MyButton color="blue" shadowSize={2}>
            Click Me
        </MyButton>

Будет преобразован в:


        React.createElement(
            MyButton,
            {color: 'blue', shadowSize: 2},
            'Click Me'
        )

Так же вы можете использовать самозакрывающуюся форму тега, если у него нет потомков:


        <div className="sidebar" />

Что будет преобразовано в:


        React.createElement(
            'div',
            {className: 'sidebar'},
            null
        )

Свойства в JSX

Есть несколько способов определения свойств в JSX:

Выражения JavaScript


        <MyComponent foo={1 + 2 + 3 + 4} />

Строковые литералы (оба выражения эквивалентны)


        <MyComponent message="hello world" />
        <MyComponent message={'hello world'} />

Свойства со стандартным значением "true" (оба выражения эквивалентны)


        <MyTextBox autocomplete />
        <MyTextBox autocomplete={true} />

Распространенные атрибуты

Если вы уже имеете объект "props" и хотите передать его в JSX, вы можете использовать "оператор распространения" : ...


        function App1() {
            return <Greeting firstName="Ben" lastName="Hector" />;
        }

        function App2() {
            const props = {firstName: 'Ben', lastName: 'Hector'};
            return <Greeting {...props} />;
        }
