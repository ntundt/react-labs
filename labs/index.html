Задание 1

Выберите и, при необходимости, установите рабочее окружение для разработки с React. В первых лабораторных работах для этого можно использовать любой текстовый редактор. В дальнейшем вам понадобится Node.JS.

Для быстрого старта можно использовать следующий шаблон:

<!DOCTYPE html>
<html>
<head>
	<meta charset="UTF-8" />
	<title>Hello World</title>
	<script src="https://unpkg.com/react@17/umd/react.production.js"></script>
	<script src="https://unpkg.com/react-dom@17/umd/react-dom.production.js"></script>
	<script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>
</head>
<body>
	<div id="root"></div>
	<script type="text/babel">
		ReactDOM.render(
			<h1>Hello, world!</h1>,
			document.getElementById('root')
		);
	</script>
</body>
</html>

Скопируйте его и убедитесь, что при открытии страницы в браузере появляется надпись "Hello, world!"

Создайте отдельный файл для скрипта. Перенесите в него весь код из исходного файла. Продемонстрируйте подключение к HTML скрипта, содержащего JSX-код.

Измените код в файле так, чтобы при открытии страницы отображалась сегодняшняя дата.
